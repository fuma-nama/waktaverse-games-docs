## 페이지 설명

`Developer Client`는 총 2개의 페이지를 제공합니다. (2025년 2월 기준)

### 로그인

`Developer Client`를 사용하려면 로그인이 필요합니다. 로그인 정보는 Waktaverse Gamse의 정보를 공유합니다.

#### 로그인 페이지

<img
  src="/images/integration-launcher/login-page.png"
  alt="로그인 페이지"
  width="100%"
/>

1.  Waktaverse Games로 사이트로 연결되는 버튼입니다.

#### 로그인 브릿지 페이지 (로그아웃 상태)

<img
  src="/images/integration-launcher/login-bridge.png"
  alt="로그인 브릿지 페이지"
  width="100%"
/>

로그인 여부를 확인 후 로그인이 안 되어 있는 경우 로그인을 진행하는 페이지입니다.

1. Google 계정을 이용한 로그인입니다.
2. Naver 계정을 이용한 로그인입니다.

#### 로그인 브릿지 페이지 (로그인 상태)

<img
  src="/images/integration-launcher/login-bridge-logged-in.avif"
  alt="로그인 브릿지 페이지 (로그인 상태)"
  width="100%"
/>

1. 상단 팝업을 호출하는 버튼입니다. (기본적으로 팝업은 자동으로 호출됩니다.)
2. 브라우저에서 `Developer Client`를 로그인 상태로 열어 주는 팝업입니다.

### 배포

게임을 배포하기 위한 페이지 입니다.

#### 메인 페이지

<img
  src="/images/integration-launcher/main.png"
  alt="메인 페이지"
  width="100%"
/>

1. 각각 `새로고침`, `최소화`, `최대화`, `닫기` 버튼입니다.
2. 각각 `Developer Client Docs`, `프로필` 버튼입니다.
3. 본인이 소속된 또는 등록한 게임들을 선택할 수 있는 영역입니다.
4. 선택된 게임의 정보를 출력하는 영역입니다.
5. 선택된 게임을 배포하기 위한 영역입니다.

#### 프로필 기능

<img
  src="/images/integration-launcher/profile.png"
  alt="프로필 기능"
  width="100%"
/>

1. 개발자 포털로 이동되는 버튼입니다.
2. 로그아웃 버튼입니다. 로그아웃 후 로그인 페이지가 열립니다.

#### 등록된 게임 리스트 기능

<img
  src="/images/integration-launcher/game-list.png"

1. 본인이 소속된 또는 등록한 게임들을 볼 수 있습니다. 하나를 선택하여 해당 게임에 대한 배포를 준비합니다.

## 배포 가이드

### 지원 OS 및 배포 형식

- Windows : 폴더 배포
- ~~Mac : 파일 배포~~ (현재 미지원)

### 배포 프로세스

#### 배포 준비

##### **1. OS 선택**

<img
  src="/images/integration-launcher/select-os.avif"
  alt="OS 선택"
  width="100%"
/>

배포한 게임이 설치될 OS를 선택합니다.

- Windows인 경우 폴더를 업로드할 수 있습니다.
- ~~Mac인 경우 파일을 업로드할 수 있습니다.~~ (Mac은 아직 지원하지 않습니다.)

##### **2. 파일 선택**

<img
  src="/images/integration-launcher/select-file.avif"
  alt="파일 선택"
  width="100%"
/>

1. 배포할 게임 폴더를 선택하는 영역입니다.
2. (선택한 게임 폴더 내) 실행 파일을 선택하는 영역입니다.
3. (선택한 게임 폴더 내) 사전 설치 파일을 추가할 수 있는 영역입니다.

##### **사전 설치 파일 (필요시 선택)**

**사전 설치 파일이란?**

게임 실행 전에 추가로 실행이 필요한 설치 파일을 말합니다. 예를 들면 C++ 재배포 패키지, .Net Framework 등이 있습니다. 해당 설치 파일들은 게임 파일에 포함되어 있어야 합니다.

<img
  src="/images/integration-launcher/pre-install.avif"
  alt="사전 설치 파일"
  width="100%"
/>

1. 사전 설치 파일을 선택하는 영역입니다.

2. 해당 설치 파일의 실행 인자입니다. 사용자가 별도로 조작하지 않아도 자동으로 설치가 진행되도록 각 설치 파일별 실행 인자를 적절히 입력해주세요.<br/><br/>
   (띄어쓰기로 구분합니다.)예) .Net 6의 경우 실행 인자를 /install /quiet /norestart 이렇게 하면 조용히 자동으로 설치가 수행됩니다.

#### 압축

게임 파일을 선택하고 정보를 입력한 뒤 압축 시작 버튼을 누르면 게임 파일을 압축하기 시작합니다.

압축이 진행되면 `게임 변경`, `배포 정보 변경`, `로그아웃` 사용이 불가능합니다. 배포가 완료되면 이후 다시 사용할 수 있습니다.

##### **1. 압축**

<img src="/images/integration-launcher/ready-zip.png" alt="압축" width="100%" />

1. 업로드할 정보와 진행상황을 볼 수 있는 토글 버튼입니다.
2. 최초 압축을 진행하는 버튼입니다.

##### **2. 압축 진행**

<img
  src="/images/integration-launcher/zip-process.png"
  alt="압축 진행"
  width="100%"
/>

##### **3. 압축 완료 및 업로드 준비**

압축이 완료되면 업로드 버튼을 눌러 최종 배포를 진행할 수 있습니다.

<img
  src="/images/integration-launcher/zip-complete.png"
  alt="압축 완료 및 업로드 준비"
  width="100%"
/>

<img src="/images/integration-launcher/upload.png" alt="업로드" width="100%" />
