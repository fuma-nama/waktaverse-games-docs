## AWS 사용 웹 게임의 CF 관리 챌린지 대응

### 보안 정책 안내

왁타버스 게임즈는 서비스 보안 강화의 일환으로 Cloudflare 관리 규칙을 통해 AWS IP 대역에 대한 접근을 제한하고 있습니다. 이는 잠재적인 봇 활동과 DDoS 공격을 사전에 방지하기 위한 조치입니다.

### 영향 및 증상

AWS 환경에서 SDK를 사용할 경우 다음과 같은 현상이 발생할 수 있습니다:

- SDK API 호출 시 404 상태 코드 수신
- API 응답이 예상과 달리 HTML 형식으로 반환

### 사용 안내

AWS 환경에서 왁타버스 게임즈 SDK를 정상적으로 사용하기 위해서는 사전 승인이 필요합니다. 이는 다음과 같은 경우에 해당합니다:

- AWS에서 호스팅되는 웹 게임
- AWS 기반 게임 서버

아래 '신청 방법' 섹션의 절차에 따라 사전 승인을 신청해 주시면, 검토 후 해당 IP에 대한 접근을 허용해 드립니다.

### 신청 방법

1. 왁타버스 게임즈 파트너즈 서버에서 협업 채널을 통하여 해당 이슈에 관해 말씀해주세요.
2. 사용하고 있는 인스턴스의 공인 IP를 아래 양식에 맞춰 협업채널에 알려주세요.

```
아래 IP 주소 차단 해제 부탁드리겠습니다.
IPv4 주소 :
IPv6 주소 :
```
