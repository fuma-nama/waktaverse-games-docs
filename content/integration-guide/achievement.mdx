## 도전과제 연동 가이드

### 도전과제 설정

게임의 도전과제를 왁겜즈 시스템과 연동하기 위해서는 먼저 도전과제를 등록해야 합니다.

1. [개발자 포탈의 게임 연동 탭](https://waktaverse.games/intro/devPortal/manage/game)에 접속합니다.
2. 해당 게임을 선택합니다.
3. 도전과제 수정 기능을 사용하여 도전과제를 등록합니다.

<Callout>
  자세한 방법은 [게임 정보 관리](/dev-portal/manage-game)를 참고해주세요.
</Callout>

### API 사용

도전과제 연동을 위해 다음 API들이 제공됩니다:

1. 사용자의 달성한 도전과제 목록 조회 API [GET /api/game-link/achieve](#)
2. 특정 도전과제 달성 API [POST /api/game-link/achieve](#)

API 사용 시 주의사항:

- 접근 토큰을 `Authorization` 헤더에 `Bearer 토큰` 형태로 설정해야 합니다.
- 자세한 API 사용법은 [도전과제 API](/api-reference/achieve)를 참고해주세요.

### 게임 지표와 도전과제 연동

게임 지표와 연결된 도전과제는 설정된 달성도를 만족하면 자동으로 잠금해제됩니다.

<Callout>
  자세한 내용은 [게임 지표 연동](/integration-guide/game-metrics)을
  참고해주세요.
</Callout>

## 도전과제 알림 연동

### Unity SDK

유니티 SDK를 사용하여 도전과제 알림을 구현하는 방법은 다음과 같습니다:

1. 알림창 영역 설정

   - 캔버스에 알림창을 띄울 영역을 추가합니다.
   - 앵커를 왼쪽 또는 오른쪽에 맞춥니다.
   - height를 화면에 맞게 조정합니다.

2. 애니메이션 설정
   - `WakgamesAchieve` 컴포넌트의 `Animation Position`을 설정합니다.
   - 옵션:
     - `Top`: 위에서 아래로 내려오는 애니메이션
     - `Bottom`: 아래에서 위로 올라가는 애니메이션
   - 기본값: `Top`

<Callout>
  게임의 UI 레이아웃에 따라 적절한 위치와 애니메이션을 선택하세요.
</Callout>

### 기타 플랫폼

다른 게임 엔진이나 플랫폼을 사용하는 경우, 왁겜즈에서 제공하는 API를 활용하여 커스텀 알림 시스템을 구현할 수 있습니다. 자세한 내용은 API 문서를 참조하세요.
