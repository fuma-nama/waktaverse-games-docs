---
title: 게임 지표 연동
---

게임 지표란 예를 들어 "플레이어의 스테이지 클리어 횟수"와 같이 인게임 활동을 정량적으로 측정한 기록을 말합니다.
이러한 지표는 사용자 데이터를 왁겜즈 서버에 저장하거나 도전과제 달성도를 추적할 때 사용할 수 있습니다.

### 지표와 도전과제 연결

게임 지표는 도전과제와 연결할 수 있습니다. 예를 들어:

- 지표: "스테이지 클리어 횟수"
- 연결 가능한 도전과제:
  - "스테이지 10번 클리어"
  - "스테이지 50번 클리어"

이렇게 연결한 경우, 해당 지표 값이 변경될 때 자동으로 관련된 도전과제들의 달성 여부가 업데이트됩니다.

### 지표 설정 절차

- 왁겜즈에 사용자별로 추적할 지표들을 등록합니다.
- [개발자 포탈의 게임 연동 탭](https://waktaverse.games/intro/devPortal/manage/game)에서 게임을 선택하고 지표 수정 기능을 사용합니다.
- 도전과제 연결이 필요한 경우, 도전과제 수정 페이지에서 관련 지표를 찾아 연결하고 목표(달성) 값을 입력합니다.

<Callout>
  자세한 방법은
  [매뉴얼](https://github.com/WakGames/Wiki/wiki/%EA%B0%9C%EB%B0%9C%EC%9E%90-%ED%8F%AC%ED%83%88#%ED%86%B5%EA%B3%84-%EC%88%98%EC%A0%95-%ED%8E%98%EC%9D%B4%EC%A7%80)을
  참고해주세요.
</Callout>

### API 사용

사용자의 현재 지표 목록을 조회하는 API와 지표 값들을 갱신하는 API가 제공됩니다.

- API 사용 시 접근 토큰을 `Authorization` 헤더에 `Bearer 토큰` 형태로 설정합니다.
- 자세한 API 사용법은 [게임 지표 연동](#) 을 참고해주세요.

<Callout type="warn">
  지표 값 갱신 시, 관련 도전과제의 달성 상태가 자동으로 업데이트될 수 있습니다.
</Callout>
