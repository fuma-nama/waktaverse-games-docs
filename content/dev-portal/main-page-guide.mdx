메인 페이지는 게임 관리를 위한 중앙 허브로서 모든 주요 기능에 대한 접근을 제공합니다.

## 주요 기능 영역

### 1. 게임 정보 표시

- 등록된 게임/어플의 기본 정보 확인
- 한눈에 볼 수 있는 주요 설정 상태

### 2. 게임 이미지 수정

> 🖼️ 게임의 시각적 요소를 관리하는 영역

- 배너 이미지 관리 (1662x430 px)
- 스크린샷 업로드 및 관리 (16:9 비율 권장)
- 표지 이미지 설정 (810x1080 px)
- 트레일러 영상 링크 관리

<Callout>이미지는 게임의 첫인상을 결정짓는 중요한 요소입니다.</Callout>

### 3. 게임 설명 수정

> 📝 게임의 정보와 메타데이터를 관리하는 영역

- 게임 소개 작성 (Markdown 지원)
- 운영체제 및 장르 설정
- 개발자/팀 정보 관리
- 스트리밍 및 다운로드 정책 설정

<Callout>명확하고 상세한 게임 설명은 사용자의 이해를 돕습니다.</Callout>

### 4. 도전과제 관리

> 🎯 게임의 성취 시스템을 구성하는 핵심 영역

- **통계 버튼**: 게임 내 통계 지표 설정
- **수정 버튼**: 도전과제 생성 및 관리

### 5. OAuth 연동 설정

> 🔐 게임과 왁겜즈 계정 연동을 위한 필수 설정

- Client ID 확인
- Callback URI 설정
- 연동 정보 관리

### 6. 팀원 추가

- 새로운 팀원에게 접근 권한 부여

<Callout type="warn">팀장 권한 필요</Callout>

### 7. 팀원 관리

- 등록된 팀원 목록 확인
- 팀원 권한 관리

<Callout type="warn">팀원 제거는 팀장 권한 필요</Callout>

## 사용 팁

1. 주요 설정 변경 전 팀원들과 상의하세요
2. 팀원 권한 부여 시 신중하게 결정하세요

## 주의사항

- 팀 관리 기능은 팀장 전용입니다
- 설정 변경은 즉시 적용될 수 있습니다
- 중요 정보는 백업을 권장합니다
