## 소개

개발자 포탈은 왁타버스 게임즈와의 게임 연동 및 정보 관리를 위한 종합 서비스 플랫폼입니다.

### 주요 기능

- 게임 정보 관리
- 이미지 및 미디어 관리
- 도전과제 시스템 구성
- OAuth 연동 설정
- 팀 구성 및 관리

## 포탈 구성

개발자 포탈은 다음과 같은 주요 섹션으로 구성되어 있습니다.

### 1. 메인 대시보드

- 게임 기본 정보 확인
- 빠른 접근 메뉴
- 팀 관리 기능

### 2. 게임 정보 관리

- 이미지 관리 (배너, 스크린샷, 표지)
- 게임 설명 및 메타 정보
- 운영체제 및 장르 설정

### 3. 도전과제 시스템

- 통계 지표 설정
- 도전과제 생성 및 관리
- 성장형 도전과제 구성

### 4. OAuth 연동

- Client ID/Secret 관리
- Callback URI 설정
- 연동 정보 설정

<Callout>각 섹션의 상세 사용법은 해당 페이지의 가이드를 참조하세요.</Callout>

## 시작하기 전에

### 주의사항

- 모든 변경사항은 저장 버튼을 클릭해야 적용됩니다.
- 팀 관리 기능은 팀장 권한이 필요합니다.
- 일부 설정은 변경 후 되돌릴 수 없습니다.

### 지원 형식

- 이미지: `.png`, `.jpg`, `.jpeg`, `.webp`
- 설명: Markdown 형식 지원
- 동영상: Youtube 임베드 링크
