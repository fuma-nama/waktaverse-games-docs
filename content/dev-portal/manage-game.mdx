### 지원 형식

<Callout>📁 **지원 확장자**: `.png`, `.jpg`, `.jpeg`, `.webp`</Callout>

### 1. 배너 이미지

- **권장 크기**: 1662x430 px
- **용도**: 스토어 메인 배너
- **팁**: 게임의 핵심 특징을 시각적으로 표현

### 2. 스크린샷 목록

### 3. 인게임 스크린샷

- **권장 비율**: 16:9
- **선택**: 변경할 스크린샷 지정 가능
- **팁**: 게임플레이를 잘 보여주는 장면 선택

### 4. 표지 이미지

- **권장 크기**: 810x1080 px
- **용도**: 스토어 대표 이미지
- **팁**: 게임의 정체성을 잘 나타내는 이미지 사용

<Callout>
  이미지에는 표기 되어 있지 않지만, 가로 버전의 이미지도 등록할 수 있는 섹션이
  마련되어 있습니다. 
  
  (SEO, Launcher 등에서 이용됩니다.)
</Callout>

### 5. 트레일러

- **형식**: YouTube 임베드 링크
- **구조**: `https://www.youtube.com/embed/{ID}`

<Callout type="warn">
  모든 변경사항은 '저장' 버튼 클릭 전까지 적용되지 않습니다!
</Callout>

### 기본 정보 설정 (1, 2, 3)

1. **게임 설명**
   - Markdown 문법 지원
   - 명확하고 상세한 정보 제공 권장

2. **시스템 요구사항**
   - 지원 운영체제 선택
   - 편집하기 버튼으로 수정

3. **장르 설정**
   - 적절한 장르 선택
   - 다중 선택 가능
