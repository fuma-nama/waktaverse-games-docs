## 왁타버스 게임즈 로고 삽입 가이드

### 개요

왁타버스 게임즈의 Partners 서비스를 이용하는 경우, 크레딧의 목적으로 게임 내에 로고를 삽입해야 합니다. 이 가이드는 로고 삽입을 위한 다양한 방법과 권장 사항을 제공합니다.

<Callout type="warn">
  **로고 삽입은 Partners 서비스 이용 약관의 일부입니다. 반드시 준수해 주시기
  바랍니다.**
</Callout>

### 로고 삽입 방법

1. 스플래시 화면

스플래시 화면은 게임 시작 시 처음 표시되는 화면으로, 로고를 효과적으로 노출할 수 있는 좋은 위치입니다.

2. 인게임 UI

스플래시 화면 사용이 어려운 경우 (예: 웹 게임, 모바일 게임 등), 다음과 같은 방법으로 로고를 삽입할 수 있습니다:

- **메인 메뉴**: 게임의 메인 메뉴 하단이나 코너에 로고 배치
- **옵션 메뉴**: 게임 설정이나 옵션 메뉴에 "About" 또는 "Credits" 섹션 추가
- **게임 종료 화면**: 게임 종료 시 표시되는 화면에 로고 포함

<Callout>
  로고가 게임의 전체적인 UI 디자인과 조화를 이루도록 배치해 주세요.
</Callout>

### 로고 사용 가이드라인

- **크기**: 로고는 쉽게 식별할 수 있는 크기여야 합니다. 너무 작지 않도록 주의해 주세요.
- **색상**: 제공된 로고의 색상을 변경하지 마세요. 필요한 경우 흑백 버전을 사용할 수 있습니다.
- **여백**: 로고 주변에 적절한 여백을 두어 다른 요소와 겹치지 않도록 해주세요.
- **변형 금지**: 로고의 비율을 변경하거나 왜곡하지 마세요.
- **회전 금지**: 로고를 회전시키거나 기울이지 마세요. 항상 원래의 방향을 유지해야 합니다.
- **배경색 주의**: 로고의 가시성을 떨어뜨리는 배경색은 피해주세요. 로고가 잘 보이도록 대비되는 배경을 선택하세요.
- **부분 사용 금지**: 로고의 일부분만 사용하는 것은 금지됩니다. 특히, 로고 타이포그래피만을 단독으로 사용하지 마세요.

<Callout>
  로고 사용에 대해 확신이 서지 않는 경우, Partners 서버를 통해 문의를
  남겨주세요.
</Callout>
